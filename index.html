LiveScratchpad by neonux

Evaluate your JavaScript functions live while editing.

Dependencies

Firefox 11 or later. Firefox Nightly recommended for all features!

Install

Get the latest add-on from http://neonux.com/devtools/LiveScratchpad/builds/
